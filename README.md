# 🏋️ Gym Workout Bot

Телеграм-бот для генерации персональных тренировочных планов с помощью Claude AI.

## Возможности

- 🤖 Генерация плана под доступное время (работает только с твоими упражнениями)
- 📋 Управление личным списком упражнений
- ⚖️ Хранение рабочих весов
- 📅 Подсчёт тренировок в неделю
- 🏗️ Структурированный порядок: базовые → изолирующие

## Установка

### 1. Клонируй / скопируй файлы проекта

### 2. Установи зависимости
```bash
pip install -r requirements.txt
```

### 3. Получи токены
- **Telegram Bot Token**: создай бота у [@BotFather](https://t.me/BotFather)
- **Anthropic API Key**: получи на [console.anthropic.com](https://console.anthropic.com)

### 4. Настрой переменные окружения
```bash
export BOT_TOKEN="твой_токен_бота"
export ANTHROPIC_API_KEY="твой_ключ_anthropic"
```

Или создай файл `.env` и добавь загрузку через `python-dotenv`.

### 5. Запусти бота
```bash
python bot.py
```

## Структура проекта

```
gym_bot/
├── bot.py                # Точка входа
├── config.py             # Конфигурация
├── database.py           # SQLite: упражнения, веса, история
├── handlers.py           # Все команды и диалоги
├── keyboards.py          # Inline и reply клавиатуры
├── states.py             # FSM состояния диалогов
├── workout_generator.py  # Генерация плана через Claude API
└── requirements.txt
```

## Использование

1. `/start` — запуск бота
2. **Добавь упражнения** через кнопку "➕ Добавить упражнение"
   - Укажи название, группу мышц, тип (базовое/изолирующее)
3. **Задай рабочие веса** через "⚖️ Рабочие веса"
4. **Начни тренировку**: выбери группу мышц → введи время → получи план!
5. После тренировки обнови веса командой `/weights`
